First install the PS3 Emulator (RPCS3):                                                                        V1.03
Official Website for download: https://rpcs3.net/download

then install the BIOS/Firmware for the PS3 with the Link below:
http://deu01.ps3.update.playstation.net/update/ps3/image/eu/2022_0510_95307e1b51d3bcc33a274db91488d29f/PS3UPDAT.PUP

then launch the emulator click on "file" and then click on "Install Firmware" then select the BIOS/Firmware File you installed and
click enter then the BIOS gets installed, then your Emulator is ready for Games
